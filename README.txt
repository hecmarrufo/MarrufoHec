Proyecto simple cuyo objetivo era mostrar los datos entregados de una base de datos .json y evaluar los datos ahí almacenados. 
Como petición adicional, se solicitó crear un botón que agregara datos dummy para probar la consistencia del diseño de la aplicación.

El proyecto se logró con HTML 5, CSS 3, JavaScript y Angularjs únicamente para el binding de datos.

En caso que desee intercambiar entre las bases de datos presentadas, ver código en scripts/script.js, dónde indico cúales líneas se deben modificar.
